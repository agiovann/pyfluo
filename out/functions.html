compute_dff¶

fluorescence.compute_dff(data, percentile=8.0, window_size=1.0, step_size=0.025, subtract_minimum=True, pad_mode='symmetric')¶

Compute delta-f-over-f

Computes the percentile-based delta-f-over-f along the 0th axis of the supplied data.

data (np.ndarray): n-dimensional data (DFF is taken over axis 0) percentile (float): percentile of data window to be taken as F0 window_size (float): size of window to determine F0, in seconds step_size (float): size of steps used to determine F0, in seconds subtract_minimum (bool): substract minimum value from data before computing pad_mode (str): mode argument for np.pad, used to specify F0 determination at start of data

Data of the same shape as input, transformed to DFF

segmentation.pca_ica(mov, components=50, batch=1000, mu=0.5, ica_func='logcosh')¶

Perform iterative PCA/ICA ROI extraction

mov (pyfluo.Movie): input movie components (int): number of independent components to return batch (int): number of pixels to load into memory simultaneously. More leads to a better fit, but requires more memory. mu (float): from 0-1. In spatiotemporal ICA, closer to 1 means more weight on spatial ica_func (str): cdf for entropy maximization in ICA

Array of shape (n,y,x) where n is number of components, and y,x correspond to shape of mov
