Note for development file system:
In order to have the docs compile properly into github pages:
1) clone the repo into the local file system
2) within the main repo directory, mkdir docs-output
3) within docs-output, git clone git@github.com:bensondaled/pyfluo.git html
4) git checkout -b gh-pages remotes/origin/gh-pages
